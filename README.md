# Personal Website

A simple, mobile-first personal website built with vanilla HTML, CSS, and JavaScript.

## Structure

- `index.html` - Main homepage with profile, projects, and blog preview
- `blog.html` - Blog posts listing page
- `blog/post-*.html` - Individual blog post pages
- `css/styles.css` - All styles (mobile-first approach)
- `js/main.js` - Minimal JavaScript

## Setup

1. Add your profile picture at `images/profile.jpg` (or update the path in `index.html`)
2. Update social media links in `index.html`:
   - Replace `https://x.com/yourusername` with your X/Twitter URL
   - Replace `https://github.com/yourusername` with your GitHub URL
3. Customize the content:
   - Update profile bio
   - Add your projects
   - Write your blog posts
4. Open `index.html` in a browser or serve with a simple HTTP server

## Features

- Mobile-first responsive design
- Clean, minimal aesthetic
- Fast loading (no dependencies)
- Easy to customize
- Static site (no build step required)
